# Retail Sales Analytics Dashboard

## Overview
The Retail Sales Analytics Dashboard is a web-based Business Intelligence system developed to transform raw retail sales data into actionable business insights.

It is designed as an applied analytics product aligned with MSc-level Business Analytics and decision-support systems, demonstrating how analytical models and data pipelines can be converted into real-world managerial tools.

## Objectives
- Enable interactive exploration of retail performance  
- Support data-driven managerial decision making  
- Provide a reusable analytical product for structurally similar datasets  
- Demonstrate applied analytics and BI system design  
- Bridge the gap between raw data processing and business strategy  

## Key Features
- Dynamic filtering by region, state, city, and date  
- Category and segment-level performance analysis  
- Time series trend visualization  
- Profit vs Sales relationship analysis  
- Hierarchical sales view using treemaps  
- Dataset structural validation before visualization  
- Smart column mapping for heterogeneous datasets  
- Downloadable filtered datasets  

## Technology Stack
- Python  
- Streamlit  
- Pandas  
- Plotly  
- Data Cleaning & Validation Pipelines  

## Dataset Compatibility
This system has been tested on real-world retail datasets including:
- Sample Superstore Dataset  
- Superstore Dataset  

Supports datasets containing:
Order Date, Region, State, City, Category, Sub-Category, Sales, Profit, Quantity, Segment
